# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added

- Initial release of the Azure SQL Database Terraform module
- SQL Server with configurable version, TLS, and connection policy
- Azure AD administrator with optional AAD-only authentication
- Managed identity support (SystemAssigned and UserAssigned)
- Elastic pool management with configurable SKU and per-database settings
- Database management with flexible SKU, collation, and retention policies
- Failover groups with automatic and manual failover modes
- Transparent data encryption with customer-managed key support
- Server-level extended auditing policy
- Security alert policy (threat detection) with email notifications
- Vulnerability assessment with recurring scans
- Firewall rules and VNet service endpoint rules
- Private endpoint connectivity with DNS zone groups
- Comprehensive examples: basic, advanced, and complete
